Overview

A full-featured photo mode system built for Project Lucian and released as a standalone Unreal Engine plugin. Players can freeze gameplay, detach the camera, and compose cinematic shots with full control over depth of field, filters, and UI visibility. The system was designed to be lightweight and easy to integrate into any Unreal Engine project.
